# RapidKey

> **An instant, minimalistic typing speed test — no account, no server, zero friction**  
> Fork of [MonkeyType](https://github.com/monkeytypegame/monkeytype) — stripped down to the pure typing experience.

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## About

RapidKey is a **desktop typing speed test** with a single goal: open it and start typing immediately. No account creation, no server setup, no Firebase - everything is stored locally in your browser.

### What Makes RapidKey Different?

- ⚡ **Zero friction** — No login. No account. Just type.
- 🖥️ **Desktop only** — Optimized for keyboard + monitor, no mobile clutter
- 💾 **100% local** — All results, settings, and history saved in your browser (localStorage)
- 🚀 **Fast** — Static site deployed on Github Page
- 🎨 **12 curated themes** — Serika Dark (red), Dracula, Nord, Monokai, and more
- 🌍 **7 languages** — English, Spanish, French, German, Portuguese, Chinese, Japanese
- **$0 infra cost** — No backend, no database, no cloud bills

### Core Features

- ✅ Clean, distraction-free typing test
- ✅ Real-time WPM, accuracy, and raw speed
- ✅ Time modes: 15s, 30s, 60s, 120s
- ✅ Word count modes: 25, 50, 100
- ✅ Last 50 results saved locally
- ✅ Personal best tracking (local)
- ✅ 12 hand-picked themes
- ✅ Smooth animated caret
- ✅ Sound effects (optional)
- ✅ Settings page (theme, font, caret style, language, sounds)
- ✅ About page with GPL attribution
